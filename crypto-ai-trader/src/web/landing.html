How a signal is born.

Three stages between a tick on the wire and an order on the book. Nothing skips a stage.

01  INGEST

Read the tape

Streams candles and order books, computes the indicator stack, and labels the current regime as trend, range or volatile.

02  REASON

Weigh the evidence

A champion and a challenger model vote. Pattern detection and Fibonacci confluence filter the noise before anything is acted on.

03  EXECUTE

Size it and send

The order is risk checked, sized to the book, and routed to the exchange. In paper mode the same path runs on virtual money.
